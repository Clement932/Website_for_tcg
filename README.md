# TCG Pocket Simulator

Un simulateur de jeu de cartes à collectionner inspiré des mécaniques modernes
de TCG mobile (progression, boosters, collection).

## Features
- Ouverture de boosters
- Gestion de collection
- Progression joueur
- Sauvegarde locale

## Stack
- HTML / CSS
- JavaScript ES Modules
- Architecture modulaire

## Objectif
Projet personnel orienté **game design & front-end**.
